# Product

## Register

product

## Users

นักเขียนนิยายไทยมืออาชีพ — คนที่เขียนเป็นอาชีพหรือ serial web novel อย่างจริงจัง ใช้ Mythoria ในสภาพแวดล้อมที่ต้องการสมาธิ บางครั้งเขียนนานหลายชั่วโมงต่อเนื่อง งานหลักคือเขียน + manage โลกนิยายที่ซับซ้อน (ตัวละคร, ไทม์ไลน์, lore, สถานที่)

## Product Purpose

Mythoria คือ story architect tool สำหรับนักเขียนไทย — ไม่ใช่แค่ editor แต่เป็น workspace ที่ช่วยให้นิยายทั้งเรื่องมีโครงสร้าง สอดคล้อง และ track ได้ จาก plot board → scene canvas → note editor → character state → lore ทุกอย่างอยู่ในที่เดียว ความสำเร็จคือนักเขียนรู้สึกว่า tool นี้ "เข้าใจ" process ของนักเขียนไทยโดยเฉพาะ

## Brand Personality

Forged, Tactical, Alive — ความรู้สึกของ workshop ที่มีชีวิต ไม่ใช่ office software มีพลังงานแบบ craftsperson ที่จริงจัง ไม่ใช่ minimalist แบบ Notion หรือ corporate แบบ Word

## Anti-references

- Notion — เรียบ clean เกินไป ไม่มี identity ของนักเขียน
- Google Docs — zero personality
- Scrivener — UI เก่า ไม่ intuitive
- Generic SaaS cream/beige AI template look

## Design Principles

1. **Forge Mode is earned, not imposed** — visual identity เด่นในที่ที่เหมาะสม (dashboard, sidebar) แต่หลีกทางให้ writing space เป็น clean canvas
2. **Tool disappears into the task** — ขณะเขียน UI ไม่ควรแย่งความสนใจ แต่ tool ควรพร้อมเสมอเมื่อต้องการ
3. **Context without interruption** — ข้อมูล (character state, plot, lore) ควรเข้าถึงได้โดยไม่ต้องออกจาก flow การเขียน
4. **Thai-first clarity** — ทุก label, state, feedback ต้องชัดเจนในภาษาไทย
5. **Density with breathing room** — tool นี้มีข้อมูลเยอะ แต่ spacing ต้องทำให้ไม่รู้สึกล้น

## Accessibility & Inclusion

- WCAG AA อย่างน้อย
- รองรับ dark mode (Forge Mode dark เป็น default สำหรับ writing session ยาว)
- Reduced motion สำหรับ animation ทุกตัว
- Thai language first, keyboard shortcuts สำคัญสำหรับ power users
